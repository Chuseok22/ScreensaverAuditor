# 릴리스 노트: 화면보호기 감사 도구 (ScreensaverAuditor) v3.0

![새 버전 소개](https://예시링크/version3_banner.png)

## 주요 변경 사항 소개 (2025년 5월)

안녕하세요! ScreensaverAuditor 팀입니다. 
v3.0 버전에서는 여러분의 소중한 피드백을 반영하여 많은 개선 사항과 새로운 기능을 추가했습니다.
특히 이번 업데이트에서는 사용자 경험을 크게 개선하는 그래픽 인터페이스를 도입하여 누구나 쉽게 사용할 수 있도록 했습니다.

### ✨ 사용자 인터페이스 혁신: GUI 모드 추가 🖥️

![GUI 스크린샷](https://예시링크/gui_screenshot.png)

- **완전히 새로워진 그래픽 인터페이스**: 직관적인 탭 기반 디자인으로 누구나 쉽게 사용할 수 있습니다.
  - `감사 설정` 탭: 날짜 범위, 사용자, 출력 파일 등 원하는 설정을 시각적으로 구성
  - `감사 결과` 탭: 수집된 화면보호기 이벤트를 표 형태로 즉시 확인
  - `도움말` 탭: 사용 방법과 문제 해결 팁을 바로 확인

- **사용 편의성 개선**:
  - **실행 모드 전환**: GUI가 기본이지만 `--console` 옵션으로 명령줄 모드 지원
  - **콘솔 창 자동 숨김**: GUI 모드에서는 콘솔 창이 표시되지 않아 깔끔한 사용자 경험 제공
  - **자동 창 크기 조정**: 다양한 화면 해상도에서 최적의 레이아웃 제공

### 🔐 더욱 편리해진 권한 관리 시스템

![UAC 알림](https://예시링크/uac_prompt.png)

- **관리자 권한 자동 요청**: 
  - 이제 애플리케이션 실행 시 자동으로 관리자 권한을 요청합니다
  - UAC 프롬프트에 한 번 '예'를 클릭하는 것만으로 모든 기능 사용 가능
  - 별도의 '관리자 권한으로 실행' 과정이 필요 없어졌습니다

- **향상된 사용자 경험**:
  - **명확한 권한 안내**: 권한이 필요한 이유와 조치 방법을 친절하게 안내
  - **상세한 오류 메시지**: 권한 문제 발생 시 해결 방법을 구체적으로 제시
  - **예외 상황 처리**: 다양한 권한 관련 예외 상황에 대한 대응 개선

### 사용자 인터페이스 개선 🖥️

- **그래픽 사용자 인터페이스(GUI) 추가**: 사용자 친화적인 인터페이스를 추가했습니다.
  - 탭 기반 인터페이스: 감사 설정, 결과 보기, 도움말 탭을 제공합니다
  - 직관적인 폼 컨트롤: 날짜 선택, 사용자 입력, 파일 경로 선택 등
  - 실시간 결과 표시: 감사 결과를 데이터 그리드로 즉시 확인 가능
  - 다양한 화면 해상도 지원: 자동 크기 조정 기능 내장

### 🏗️ 아키텍처 현대화 및 확장성 향상

![코드 구조](https://예시링크/architecture.png)

#### 클린 아키텍처 도입

기존의 모놀리식 구조에서 **MVVM(Model-View-ViewModel) 패턴**을 도입하여 코드의 가독성, 유지보수성 및 테스트 용이성을 크게 향상했습니다:

| 계층 | 구성 요소 | 설명 |
|-----|---------|------|
| **모델(Model)** | `Models/` 폴더 | 애플리케이션의 핵심 데이터 구조 정의 |
| | - `ScreensaverEvent.cs` | 화면보호기 이벤트의 데이터 모델 |
| | - `ScreensaverUsage.cs` | 사용 통계 계산을 위한 집계 모델 |
| | - `CommandLineOptions.cs` | 명령줄 옵션 처리를 위한 모델 |
| **서비스(Services)** | `Services/` 폴더 | 핵심 비즈니스 로직 처리 |
| | - `ScreensaverAuditorService.cs` | 이벤트 로그 수집 및 분석 서비스 |
| | - `ExcelExporter.cs` | Excel 보고서 생성 서비스 |
| **뷰모델(ViewModel)** | `ViewModels/` 폴더 | 뷰와 모델 간의 중재자 |
| | - `ScreensaverAuditorViewModel.cs` | UI 상태 관리 및 명령 처리 |
| **뷰(View)** | `UI/` 폴더 | 사용자 인터페이스 컴포넌트 |
| | - `MainForm.cs` | 메인 애플리케이션 창 및 탭 컨테이너 |
| **유틸리티** | `Utils/` 폴더 | 공통 기능 유틸리티 클래스 |
| | - `CommandLineParser.cs` | 명령줄 인수 처리 |
| | - `ConsoleHelper.cs` | 콘솔 출력 유틸리티 |
| | - `FormResizer.cs` | UI 화면 크기 자동 조정 |
| | - `IconConverter.cs` | 아이콘 리소스 처리 |

### 🆕 새로운 기능 및 성능 향상

![새 기능](https://예시링크/new_features.png)

#### 명령줄 기능 강화

- **원클릭 감사 지원**: `--audit` 명령어를 통해 기본 설정으로 즉시 감사 실행
  ```bash
  # 단 한 번의 명령으로 최근 7일간 감사 실행
  ScreensaverAuditor.exe --audit
  
  # GUI 모드를 유지하면서 자동 감사 수행
  ScreensaverAuditor.exe --auto-audit
  ```

- **배치 처리 지원**: 여러 명령을 하나의 호출로 결합
  ```bash
  # 정책 설정, 감사 실행, 파일 저장을 한 번에 처리
  ScreensaverAuditor.exe --console --batch="enable-policy,audit,export"
  ```

#### 데이터 처리 개선

- **고급 JSON 파서 구현**:
  - 📋 **다양한 JSON 형식 지원**: PowerShell 및 Windows 이벤트 로그 JSON 출력 파싱
  - ⏱️ **날짜 형식 자동 변환**: Microsoft JSON 날짜 형식(`/Date(타임스탬프)/`) 자동 처리
  - 🔍 **중첩 JSON 구조 지원**: 복잡한 이벤트 메타데이터 추출 가능
  - 🚀 **성능 최적화**: 메모리 효율적인 JSON 스트림 처리

- **실시간 데이터 모니터링**:
  - 📊 **라이브 모드**: 화면보호기 활동을 실시간으로 모니터링하는 새로운 모드 추가
  - 🔔 **알림 지원**: 특정 패턴 발견 시 Windows 알림 표시
  - 📈 **추세 분석**: 일일/주간/월간 활동 패턴을 자동으로 비교 분석

### 📊 향상된 데이터 분석 및 보고서 기능

#### 풍부해진 데이터 필드

이제 더 상세한 분석을 위한 다양한 필드를 제공합니다:

| 추가된 필드 | 설명 | 활용 방법 |
|-----------|------|---------|
| **이벤트 유형** | 화면보호기 시작/종료 구분 | 필터링으로 특정 유형만 빠르게 확인 가능 |
| **작업 유형** | Windows 이벤트의 TaskDisplayName | 세부 이벤트 유형 분석에 활용 |
| **활동 ID** | 연관된 이벤트 그룹화를 위한 식별자 | 동일 세션의 이벤트 추적에 유용 |
| **이벤트 제공자** | 이벤트를 생성한 Windows 구성 요소 | 다양한 보안 이벤트 소스 분석 |
| **감사 결과** | 성공/실패 등 이벤트의 결과 정보 | 이벤트 처리 상태 확인 |

#### 시각적 개선 및 사용자 경험

- **자동 셀 서식**:
  - 🟢 **화면보호기 시작**: 초록색 배경으로 쉽게 구분
  - 🔴 **화면보호기 종료**: 빨간색 배경으로 명확히 표시
  - ⚠️ **이상 패턴**: 비정상적인 지속 시간은 노란색 경고 표시

- **고급 데이터 시각화**:
  - 📈 **자동 생성 피벗 테이블**: 사용자별/시간대별 요약 통계
  - 📊 **내장 차트**: 주요 지표를 시각적으로 표현하는 차트 시트
  - 🔍 **고급 필터**: 원클릭으로 모든 열에 고급 필터링 적용 가능

- **데이터 내보내기 개선**:
  - 💾 **다양한 형식 지원**: XLSX 외에도 CSV, PDF 형식 지원
  - 🖨️ **인쇄 최적화**: 보고서 형태로 깔끔하게 인쇄되도록 서식 개선
  - 📱 **모바일 호환성**: 모바일 기기에서도 보기 좋게 최적화

## 🚀 업그레이드 안내 및 마이그레이션

![업그레이드 가이드](https://예시링크/upgrade_guide.png)

### 쉬운 업그레이드
- **즉시 사용 가능**: 설치 후 별도의 설정 변경 없이 바로 사용할 수 있습니다.
- **기존 설정 보존**: 이전 버전에서 활성화했던 감사 정책은 그대로 유지됩니다.
- **자동 업데이트 확인**: 시작 시 최신 버전을 자동으로 확인하고 업데이트를 안내합니다.

### 마이그레이션 팁
- **이전 보고서 호환성**: 이전 버전에서 생성된 Excel 보고서는 새 버전에서도 호환됩니다.
- **설정 파일 위치**: 사용자 설정 파일은 `%APPDATA%\ScreensaverAuditor\` 폴더에 그대로 유지됩니다.
- **명령줄 스크립트**: 기존 명령줄 스크립트는 `--console` 옵션만 추가하면 그대로 사용 가능합니다.
